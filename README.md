# 🏥 Healthcare Analytics Dashboard

An interactive Power BI dashboard designed to analyze healthcare data and provide actionable insights into patient demographics, admissions, billing, insurance providers, medical conditions, medications, and hospital performance.

---

## 📌 Project Overview

This project demonstrates how Power BI can be used to transform raw healthcare data into meaningful business insights. The dashboard enables users to monitor key performance indicators (KPIs), identify trends, and make data-driven decisions through interactive visualizations.

---

## 🎯 Objectives

- Analyze patient admissions and discharge trends.
- Monitor total healthcare billing and average billing per patient.
- Identify the most common medical conditions.
- Compare billing across insurance providers.
- Analyze patient demographics by age and gender.
- Measure the average length of hospital stay.
- Enable interactive filtering for better decision-making.

---

## 📊 Dashboard Features

### KPI Cards
- 👥 Total Patients
- 💰 Total Billing
- 👤 Average Age
- 🏥 Average Length of Stay
- 💵 Average Billing

### Interactive Visualizations
- Patient Admissions by Month
- Patients by Medical Condition
- Billing by Insurance Provider
- Patients by Admission Type
- Patients by Medication
- Test Results Analysis
- Age Group Distribution
- Gender Distribution

### Interactive Filters
- Gender
- Year
- Insurance Provider
- Medical Condition
